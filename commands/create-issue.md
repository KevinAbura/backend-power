# bp:create-issue

## Purpose

Create or enrich a GitHub Issue from design and planning context.

## Inputs

- design reference
- backend scope summary
- acceptance criteria
- verification notes

## Outputs

- GitHub Issue with design linkage
- backend scope sections
- reference implementation placeholders

## Usage

Use this command after requirement extraction and before implementation.

## Notes

Issues created by this command should preserve traceability and set up later implementation and review work.
