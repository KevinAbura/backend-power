# bp:find-reference

## Purpose

Retrieve similar implementations from the existing repository before planning or coding.

## Inputs

- issue or story context
- domain keywords
- optional repository paths

## Outputs

- similar endpoints
- similar services
- similar migrations
- similar tests
- repository style guidance

## Usage

Run this command before implementation or review.

## Notes

This command is mandatory for repository-grounded generation. Backend Power should prefer existing patterns over invention.
