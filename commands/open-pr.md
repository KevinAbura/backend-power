# bp:open-pr

## Purpose

Create a pull request with linked work item, design reference, and verification summary.

## Inputs

- head branch
- base branch
- linked issue
- change summary
- verification summary

## Outputs

- pull request
- structured PR description

## Usage

Use this command after implementation and test hooks complete.

## Notes

PRs should highlight reference patterns used and call out any deviations from repository conventions.
